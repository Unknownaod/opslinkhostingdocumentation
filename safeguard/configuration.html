Configuration

Setting up OpsLink Safeguard is simple. You can configure everything either via Discord commands or through the dashboard (coming soon).

Step-by-Step Setup

  1. Invite the Safeguard bot to your Discord server.
  2. Ensure the bot has Administrator and View Audit Log permissions.
  3. Run /setup to initialize Safeguard‚Äôs data structure.
  4. Set your logging channels using /setlog.
  5. Enable auto-mod rules with /automod enable.

Customizing Behavior

Each module can be toggled on/off using /config commands. You can configure thresholds, punishment types, and notification channels per module.
