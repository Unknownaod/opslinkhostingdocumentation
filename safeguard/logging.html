Logging & Audit System

Safeguard keeps comprehensive audit logs for every major server event, ensuring transparency and accountability within your Discord community.

Default Logs

  • Message Deletion / Edits
  • Role and Permission Changes
  • User Bans, Kicks, and Timeouts
  • Join / Leave Events
  • Command Executions

Custom Channels

Use /setlog [type] [channel] to direct logs to specific channels (e.g., /setlog bans #admin-logs).
