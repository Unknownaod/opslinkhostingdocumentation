Control Panel Overview

The OpsLink Control Panel is built on Pterodactyl for performance and simplicity, letting you manage game and bot servers from one interface.

Server Management

  • Start, stop, and restart servers in real time.
  • Monitor RAM, CPU, and network usage.
  • Access your server console for live debugging.

File Manager

Navigate your files directly in the browser, upload ZIPs, and edit scripts inline. You can also access your instance via SFTP for faster bulk operations.

Scheduling

Automate restarts, backups, or updates with the built-in task scheduler. Simply go to Schedules ‚Üí New Schedule and define your timing.
