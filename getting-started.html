Getting Started

Welcome to OpsLink Hosting! Follow these steps to launch your first game or bot server.

1. Access the Panel

Go to panel.opslinkcad.com and log in with your provided credentials.

2. Create Your Server

Once logged in, click “Create Server” and choose your desired game or bot environment.

3. Configure Startup Settings

Use the environment variables to set your PORT, SERVER_KEY, and other configurations.

4. Start Your Server

Click the Start button in the console view. You can access logs, SFTP, and real-time resource usage.
