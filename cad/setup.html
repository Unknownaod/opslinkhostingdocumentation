Setup Guide

This page explains how to get started with OpsLink CAD for your community or department.

1. Access Your CAD Instance

Each community is provided a unique subdomain (e.g., yourcommunity.opslinkcad.com). Contact OpsLink Systems to request your deployment.

2. Configure Departments

Visit the Admin Dashboard ‚Üí Departments ‚Üí Add Department. Assign command roles and set department identifiers (PD, FD, CIV, etc).

3. Add Users

  1. Open the Users tab.
  2. Click Add User.
  3. Assign rank and permissions (e.g., Dispatcher, Officer).
  4. Send invite link to your members.
